Authentication API Documentation

Interact with the Authentication API endpoints hosted at https://shoppica-backend.onrender.com/api

1. Add New User

Endpoint: POST /users

Description: Creates a new user in the system. No authentication required.

2. Login

Endpoint: POST /login

Description: Authenticates a user and establishes a session.

3. Get Current User

Endpoint: GET /me

Description: Retrieves details of the currently authenticated user.

4. Logout

Endpoint: POST /logout

Description: Logs out the current user by clearing the session.
